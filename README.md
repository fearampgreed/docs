# NACE Billing Documentation

**Audience: Billing team.** This site explains NAPA, how to use the Billing app, and how to handle invoices and disputes. It is not written for developers.

---

*For maintainers: This site is built with [Mintlify](https://mintlify.com). To preview locally: `npm i -g mint` then `mint dev` in this folder. Deploy via [Mintlify Dashboard](https://dashboard.mintlify.com) (connect repo, set docs root to `docs`). MCP URL after deploy: `https://<your-subdomain>.mintlify.com/mcp`.*
